Installation

  1. Open terminal of your choice and type
git clone https://github.com/mern-stack/mern.git <Your-Project-Name>

  1. CD into the <Your-Project-Name>/backend directory and rename .env.example to .env

  2. Enter PORT number to port of your choice, for example PORT=5000.

     This will be used for running server on the specified port

  3. Enter a secure random string to KEY= in .env

    You can get a secure key from Generate Random Key

    It Is recommended to generate 256 bit Key

  4. Now run npm run setup in your terminal and we will set everything for you.
