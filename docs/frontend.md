Frontend is Redux Powered react app.

# Directory Structure

```
root-directory
├── fontend // all your client related stuff lives here
│   ├── src 
│   │   ├─ actions
│   │   │   └── controller
│   │   ├── common // some common elements lives here
│   │   ├── components // all components are present here
│   │   ├── layouts // layouts of project
│   │   ├── reducers
│   │   ├── utils // common and utilities lives here
│   │   └── providers    
│   └── public // static files will be stores here
└── backend // all your server related stuff goes here

```
